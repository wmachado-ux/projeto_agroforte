# Projeto AgroFuturo

Projeto submetido para o concurso Agrinho.

Uma plataforma educacional interativa sobre Agricultura Sustentável, Agricultura 4.0, inovação tecnológica e preservação ambiental, desenvolvida para transformar o aprendizado em uma experiência dinâmica, visual e gamificada. O projeto combina simuladores, gráficos, desafios, quiz, conquistas e certificado de conclusão para estimular o protagonismo dos estudantes.

---

## Sobre o Projeto

O site apresenta uma jornada educativa onde o usuário assume o papel de gestor de uma fazenda do futuro, tomando decisões relacionadas ao uso de recursos naturais, tecnologia e sustentabilidade.
O objetivo é demonstrar como tecnologia e sustentabilidade podem caminhar juntas para garantir produtividade e preservação ambiental.
